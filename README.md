# dvm-engine

Dvm-engine is a free and open-source, lightweight, minimal 3D engine boilerplate.
It provides very little features but sets a starting point for anyone who would like to start building a game.

## Features

- [X] Multiplatform
  - [X] Windows
  - [X] Linux
  - [X] MacOS (untested)
- [X] OBJ Mesh import
- [X] FPS camera
- [X] GUI
- [X] Lighting
  - [X] Diffuse + Specular
  - [X] Directional light
  - [X] Point lights
  - [ ] Area lights
  - [ ] Ambient Occlusion
  - [ ] Shadows
  - [ ] Environment textures
- [ ] Material system
  - [X] Texture loading
  - [ ] PBR
- [X] ECS
